## Dependencies

```bash
$ bin/bundle install
```

## Running

cd into this dir and start the Rails 6 server on port 4000:

```bash
$ bin/rails s --port 4000
```

Start the [React frontend](https://github.com/iq9/users-frontend-react).

It will be browseable on port 3000: http://localhost:3000/
